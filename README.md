# Чек-лист визуальной самооценки износа здания медицинской организации — PWA v18 iOS

## Как тестировать на iPhone
1. Разместить всю папку на HTTPS-хостинге.
2. Открыть ссылку `index.html` именно в Safari.
3. Проверить, что в блоке результата написано: «Расчёт запущен...».
4. Выбрать баллы в блоке 3. Индекс должен обновиться сразу; дополнительно включён автопересчёт каждые 0,5 секунды.
5. Safari: «Поделиться» → «Добавить на экран “Домой”».

Локальное открытие HTML из «Файлы», Telegram или почты не является полноценным PWA-режимом и может блокировать JavaScript, камеру, кэширование или печать.
