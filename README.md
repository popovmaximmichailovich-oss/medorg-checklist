# Чек-лист визуальной самооценки износа здания медицинской организации — PWA v20

## Обновление на GitHub Pages
1. Распакуйте архив.
2. В репозитории GitHub откройте вкладку Code.
3. Нажмите Add file → Upload files.
4. Загрузите содержимое архива с заменой файлов: index.html, manifest.webmanifest, service-worker.js, README.md, icons/icon-192.png, icons/icon-512.png.
5. Нажмите Commit changes.
6. Откройте сайт с параметром ?v=20.
7. На iPhone удалите старую иконку PWA с экрана «Домой», затем откройте ссылку в Safari и добавьте заново.
