# Чек-лист визуальной самооценки износа здания медицинской организации — PWA v19

Версия с комплексным отчётом для печати/PDF и фотофиксацией.

## Обновление на GitHub Pages
1. Откройте репозиторий на GitHub.
2. Перейдите во вкладку Code.
3. Нажмите Add file → Upload files.
4. Загрузите содержимое этой папки: index.html, manifest.webmanifest, service-worker.js и папку icons.
5. При запросе замены файлов подтвердите загрузку.
6. Внизу нажмите Commit changes.
7. Подождите 1–5 минут.
8. На iPhone откройте сайт в Safari. Если открывается старая версия: обновите страницу, закройте и снова откройте PWA с экрана Домой. При необходимости удалите и заново добавьте приложение на экран Домой.
